STEP7: Click on Close

STEP8: To Check whether JAVA is installed or not,Open command prompt then enter command as javac.
If it gives message as follows,which means JAVA is not installed,don't worry close it.

STEP9: Open control panel and then click on system and security

STEP10: Click on System

STEP11: Click on Advanced system settings

STEP12: Click on Environmental Variables

STEP13: Double Click on path

STEP14: Now path appears like this,put ; after the provided default path

STEP15: Go to drive in which your java has installed
Mostly it follows the following path :localdisk(c)-->program files-->java-->jdk1.8.0_111-->bin
Now copy the link

STEP16: Paste the copied path after ;(refer step14)
Now put another ; after the copied link

STEP17: localdisk(c)-->program files-->java-->jre1.8.0_111-->bin
copy the link

STEP18: Again paste the copied path after ; and click Ok

STEP19: Click on New Button

STEP20: Enter Variable Name as JAVA_HOME

STEP21: localdisk(c)-->program files-->java-->jdk1.8.0_111-->bin
Again copy the link

STEP22: Paste the copied link here and then click OK

STEP23: Click on OK

STEP24: Again click on OK

STEP25: Open command prompt and then enter the command as javac

STEP26: If command prompt looks like this then your JAVA Installation process is successful
